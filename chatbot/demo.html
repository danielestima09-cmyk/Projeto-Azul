Como usar

Clique no botão flutuante 💬 (canto inferior direito) para abrir o chat. Pergunte, por exemplo: “Por que o projeto foi bloqueado?”, “Quais ajustes preciso fazer?”, “Como está o compliance?” ou tente “aprove esse projeto para mim” (para ver a recusa segura).

Este preview usa window.ABOS_CHAT_TRANSPORT para simular respostas offline. Em produção, o mesmo chatbot.js chama /api/chat (Gemini via OpenRouter) sem alterações.
