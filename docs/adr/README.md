# ADRs — Architecture Decision Records

Registro curto das decisões de arquitetura importantes. Cada arquivo segue o formato
**Contexto · Decisão · Alternativas · Consequências**.

> ADR (em uma frase): um documento curto que registra *por que* uma decisão foi tomada, para que
> no futuro ninguém precise adivinhar o motivo.
